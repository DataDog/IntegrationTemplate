Overview

Split is the leading platform for controlled rollouts, helping businesses of all sizes deliver exceptional user experiences—and mitigate risk—by providing an easy, secure way to target features to customers. Companies like WePay, Segment and thredUP rely on Split to safely introduce and test new features, and draw insights on their use. Out of the box, Split provides:

  • Feature flags in Java, JavaScript, NodeJS, .NET, PHP, Python, Ruby and Ruby-on-Rails
  • Open source, language-specific SDKs
  • Flexible feature roll out capabilities, by-percentage, by-whitelist, or by-attribute
  • Robust feature-targeting capabilities, for any attribute you can define.
  • Enterprise-grade security (no need to ever share user information with Split)
  • A kill switch, to instantly halt and revert any feature, without touching code
  • A beautiful web console to manage features rollouts
  • Integrations with the DevOps tools your team is already using, like Slack, JIRA, and of course, Datadog

Configuration

In Datadog:

  1. Go to Integrations and click APIs.
  2. Click Create API Key.
  3. Copy the API key you just created.

In Split:

  1. Go to Admin Settings and click Integrations and navigate to the Marketplace. Click Add next to Datadog.
  2. Paste the API Key you copied in step 3 and click Save.
  3. Copy the API key you just created.

Split data should now be flowing into Datadog.
