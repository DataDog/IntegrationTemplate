Icinga stats

Icinga is a monitoring system which checks the availability of your resources, notifies users of outages and provides extensive BI data.

Overview

Get your alert statistics from icinga to Datadog

  • Show your Systemalerts on a board

  • Visualize a alert history

Configuration

To capture metrics you need to install and configure the Datadog Agent.

  1. Configure the Agent to connect to Icinga
    Edit conf.d/icingastats.yaml 
                
init_config:

instances:
- site: "SITENAME"
  cgiurl: "https://YOUR.MONITORINGHOST.ch/cgi-bin/icinga"
  username: "YOURICINGAUSER"
  password: "YOURICINGAPASS"
