RBLTracker

RBLTracker provides easy-to-use, real-time blacklist monitoring, for your email, website, and social media.

Overview

Connect your RBLTracker account to Datadog to:

  • Push listing events from RBLTracker to your dashboard.
  • Push de-listing events from RBLTracker to your dashboard.

Configuration

Setting up RBLTracker using webhooks:

  1. In Datadog, create a new Application Key called "RBLTracker".
  2. In RBLTracker, create a new Datadog contact type from the Manage -> Contacts section of the RBLTracker portal.
  3. Provide both the Datadog API Key and Application Key.
  4. (optional) adjust the contact schedule for this new contact.

RBLTracker will send listing and delisting alerts to your Datadog events dashboard. Click here for a full integration guide.
