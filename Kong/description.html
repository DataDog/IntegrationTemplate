Kong

Secure, Manage & Extend your APIs and Microservices

Overview

Connect Kong to Datadog to:

  • Visualize Kong performance data
  • Correlate the performance of Kong with the rest of your applications

Configuration

To capture Kong metrics you need to install the Datadog Agent.

  1. Configure the Agent to connect to Kong
    Edit conf.d/kong.yaml 
                
              init_config:

              instances:
              # For every instance, you have an `kong_status_url` and (optionally)
              # a list of tags.

                -   kong_status_url: http://example.com:8001/status/
                    tags:
                        -   instance:foo

                -   kong_status_url: http://example2.com:8001/status/
                    tags:
                        -   instance:bar
  2. Restart the Agent
  3. Execute the info command and verify that the integration check has passed
  4. Select Integrations, and then select ‘Connect Datadog’
