Rollbar

Rollbar helps development teams quickly detect, diagnose and defeat application errors.

Overview

Rollbar helps development teams detect, diagnose and defeat application errors. View exceptions from all of your languages, frameworks, platforms and environments in one place. Integrating Datadog with Rollbar will send exceptions, errors and code deployments to your Datadog event stream, in real time.

Connect Rollbar to Datadog to

  • Get notified of exceptions, errors, code deployments in your event stream
  • Filter notifications by severity, environment, host, users and more
  • Search for exceptions in your graphs
  • Discuss exceptions with your team
  • Spend less time debugging issues

Configuration

Configuration is per-project in Rollbar.

  1. In Rollbar, go to the Notification settings page for a Project: Dashboard → Settings → Notifications → Datadog
  2. On Datadog, head to the APIs page to get your API Key (copy to your clipboard): Integrations → APIs
  3. Add your Datadog API key (${api_key}) in Rollbar
  4. Click Enable Datadog Integration

Done. Every time an exception occurs, it will appear in your event stream.
