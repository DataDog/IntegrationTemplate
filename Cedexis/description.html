Manage Openmix traffic with Datadog monitors and alerts.

Overview

Connect Datadog to Cedexis Openmix to:

  • Manage traffic using Datadog monitors to measure the health of routing endpoints
  • Trigger immediate traffic changes from Datadog alerts
Cedexis Openmix manages global application and content delivery - it is a control plane for delivery that is integrated with key ecosystem services, including Datadog. It receives and processes data inputs to feed user-configurable algorithms, which make traffic routing and load balancing decisions that are communicated into applications via DNS or an HTTP API. Because Openmix delivery optimization is based on the real-time health of infrastructure endpoints, it optimizes user experience; maximizes performance; and provides granular control for organizations according to their unique business rules.
